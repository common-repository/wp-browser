== Description ==

WP-Browser will help for your users easily and quickly browse through your categories and articles, while your hosting will be spending fewer resources and web traffic. Even with heavy traffic, server resources will be saved by caching the all functional elements. Also WP-Browser will help you get a better earning on advertising.

* Multi-functional navigation bar with using AJAX.
* Allows you to preview all contents of category on one page.
* Saves traffic and server resources. 
* Comfortable Navigation
* A set of configuration options
* Automatic installation
* Economy traffic and server resources
* The increase in income from advertising
* Automatic caching

== Installation ==

1. Upload `wp-browser` folder or unzip `wp-browser.zip` archive to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the parameters of the plugin and the plugin is ready to use!
4. To run the plugin, you don't need to edit your theme. The navigation bar is inserted automatically.
